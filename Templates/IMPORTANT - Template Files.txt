ВАЖНАЯ ИНФОРМАЦИЯ ДЛЯ ПОЛЬЗОВАТЕЛЕЙ ВЕРСИЙ С НЕСКОЛЬКИМИ ПАКЕТАМИ ЛОКАЛИЗАЦИИ

Применимо в следующих случаях.
 — При установке версий с несколькими пакетами локализации с сохранением папки установки шаблонов по умолчанию (C:\Users\Public\Documents\Autodesk\Inventor release year\Templates).
 — При хранении и применении пользовательских шаблонов в папке C:\Users\Public\Documents\Autodesk\Inventor release year\Templates.

В Inventor имеется поддержка пакетов локализации для многих версий. Благодаря этому можно устанавливать несколько языковых версий в составе одной установки Inventor и переключаться между ними с помощью ярлыка в меню «Пуск» (в котором используется параметр /language).  

Несмотря на то, что с пакетами локализации устанавливаются локализованные файлы шаблонов, при запуске разных языковых версий Inventor используемые по умолчанию шаблоны не меняются. 

В Inventor 2020 все пакеты локализации устанавливаются и обрабатываются одинаково. Больше не нужно копировать шаблоны в родительскую папку, так как при изменении языка путь к файлу шаблона обновляется. Это упрощает использование и обслуживание всех шаблонов (как используемых по умолчанию, так и пользовательских).
